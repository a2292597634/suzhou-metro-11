📋 导出步骤

  1. 确保本地服务器已启动（python -m http.server 8080）
  2. 在项目目录打开命令行（终端/PowerShell）
  3. 运行导出命令（已安装依赖）：
    node tools/export-hd.js
  4. 导出的 PNG 文件保存在 output/ 文件夹

💡 快速命令

# 导出全部规格
npm run export-hd

# 仅导出 4K（推荐，A1海报）
npm run export-4k

# 仅导出 8K（最高清）
npm run export-8k
