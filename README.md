# 苏州轨道交通11号线商业开发作战图

> 一个可编辑、可保存、可高清打印的苏州地铁11号线商业数据可视化作战图网站。

## 在线访问

部署后即可通过公网链接访问。

## 项目特点

- **21:9 宽屏比例** - 专为作战图上墙设计，适配大屏和打印输出
- **数据实时编辑** - 单击数字直接修改，双击卡片打开完整编辑器
- **本地自动保存** - 所有修改自动写入浏览器 localStorage，刷新不丢失
- **数据导入导出** - 支持 JSON 格式，便于团队协作和备份
- **高清打印优化** - 内置 `@media print` 样式，一键打印即可上墙
- **28个站点完整覆盖** - 包含主线、支线全部站点及商业价值分级

## 使用方法

### 本地打开
直接用浏览器打开 `index.html` 即可。

### 数据编辑
- **单击**站点卡片上的任意数字（商铺/已出租/空置/装修中）→ 直接输入修改
- **双击**站点卡片 → 弹出完整编辑器，可修改名称、等级、坐标、位置等
- **单击**顶部统计面板的数字 → 修改全局统计

### 打印作战图
点击右上角「🖨️ 打印作战图」按钮，浏览器进入打印预览：
- 建议纸张：**A3 横向** 或自定义大幅面
- 打印设置：开启「背景图形」，关闭页眉页脚

### 数据备份
- 点击「📥 导出数据」生成 JSON 文件
- 点击「📤 导入数据」恢复之前的数据

## 部署方法

### 方案一：GitHub Pages（推荐，免费永久）

1. 注册/登录 [GitHub](https://github.com)
2. 新建仓库，名称如 `suzhou-metro-11`
3. 上传本项目所有文件到仓库
4. 进入 Settings → Pages → Source 选择 `main` 分支
5. 等待1-2分钟，获得 `https://你的用户名.github.io/suzhou-metro-11` 链接

### 方案二：Netlify Drop（最简单）

1. 打开 [netlify.com](https://netlify.com)
2. 把项目文件夹直接拖拽到网页上
3. 自动生成随机域名，可立即访问

### 方案三：Vercel

1. 注册 [vercel.com](https://vercel.com)
2. 导入 GitHub 仓库，一键部署

### 方案四：Gitee Pages（国内访问快）

1. 注册 [gitee.com](https://gitee.com)
2. 创建仓库并上传代码
3. 开启「服务 → Gitee Pages」
4. 获得国内访问更快的链接
